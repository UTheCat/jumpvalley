Class that runs a tween by incrementing steps at a specified time interval. ~\newline
 Note that while the time interval should mostly be fixed, there will be slight differences in the actual time between each step. ~\newline
 To account for this, the Stopwatch class is used to run the tween based on a timestamp. Therefore, the aforementioned differences shouldn\textquotesingle{}t add up and cause the tween to finish noticeably later than expected for longer tweens.
