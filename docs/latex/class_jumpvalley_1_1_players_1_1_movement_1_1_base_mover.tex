\doxysubsection{Detailed Description}
This is the base class that provides a player\textquotesingle{}s character the ability to move in different directions, jump, and climb. It serves mainly as a controller interface for developers to build on. ~\newline
 The design of this takes lots of inspiration from Roblox\textquotesingle{}s Player\+Module.
