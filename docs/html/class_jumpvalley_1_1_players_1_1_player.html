This class represents a player who is playing Jumpvalley.

Detailed Description

This class represents a player who is playing Jumpvalley.
The class contains some of the basic components that allow Jumpvalley to function for the player, such as:

  • Their music player
  • The Controller instance that allows them to control their character
  • The Camera instance that allows them to control their camera
  • Their primary GUI node
