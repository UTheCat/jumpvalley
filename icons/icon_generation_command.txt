Using ImageMagick, the command to generate the icon (.ico) file from Terminal is:

magick convert app_icon_16px.png app_icon_32px.png app_icon_48px.png app_icon_64px.png app_icon_128px.png app_icon_256px.png -colors 256 app_icon.ico

This might also work:
magick convert app_icon.ico -define icon:auto-resize

Or this (from testing, this gives the best result):
magick convert app_icon_16px.png app_icon_32px.png app_icon_48px.png app_icon_64px.png app_icon_128px.png app_icon_256px.png app_icon.ico
